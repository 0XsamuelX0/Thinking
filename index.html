Sobre Nosotros

Somos un equipo enfocado en crear experiencias educativas únicas para nuestros estudiantes.

Misión

Garantizamos que nuestro blog de aprendizaje sea la indicada para aquellas personas que quieran aprender de manera autodidacta los conceptos básicos de programación. Con el fin de brindar una calidad educativa avanzada para el fortalecimiento del pensamiento computacional de la institución educativa, Jose Acebedo y Gomez. Los usuarios podrán disponer de una amplia información sobre; HTML, CSS, JavaScript y pensamiento computacional, además de disponer de un quiz de preguntas para fortalecer lo aprendido.

Acerca de

Nuestra página web se enfoca en aquellos estudiantes que se sienten interesados en ingresar a la media técnica de software;

con nuestra página web obtendrán información sobre los conceptos más básicos de programación donde explica, soluciona y enseña a un grado de dificultad que se maneja en la media técnica del grado.

Servicios

Aprendizaje en línea flexible, dinámico y relevante para cualquier persona que quiera aprender programación.

HTML

Lenguaje de marcado HTML5 utilizado en el desarrollo web para crear el contenido y la estructura de una página web.

CSS

Lenguaje de hojas de estilos utilizado en el diseño web para dotar de aspecto visual a una página web.

JAVASCRIPT

Lenguaje de programación interpretado utilizado en el lado del cliente (front-end) para dotar de mejoras dinámicas e interactivas a las páginas web.

Pensamiento Computacional

Brindamos la información más amplia sobre el pensamiento lógico de un programador.
